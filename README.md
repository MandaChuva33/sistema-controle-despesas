# Sistema de Controle de Despesas

## Autor
Pedro Pourchet - RA: 206454-25

## Descrição do Projeto
Este projeto implementa um Sistema de Controle de Despesas em Java, seguindo os princípios da Programação Orientada a Objetos (POO) e utilizando arquivos de texto para persistência de dados. O sistema atende aos requisitos de Herança, Polimorfismo, Sobrecarga, Sobrescrita, Métodos e Atributos Estáticos, e Criptografia de Senhas.

## Estrutura do Repositório

| Arquivo/Diretório | Descrição |
| :--- | :--- |
| `src/Main.java` | Contém a lógica principal do menu e a interação com o usuário. |
| `src/Despesa.java` | Contém as classes de modelo (`TipoDespesa`, `Usuario`) e as classes de POO (`Pagavel`, `Despesa`, `DespesaAlimentacao`). |
| `src/GerenciadorDespesas.java` | Contém as classes de acesso a dados (DAOs) para `TipoDespesa`, `Usuario` e `Despesa`. |
| `docs/` | Diretório para documentação. |
| `docs/README.md` | Estratégia de construção do sistema. |
| `docs/CHANGELOG.md` | Histórico de versões e commits. |
| `docs/CLASSES.md` | Documentação detalhada das classes e conceitos de POO aplicados. |
| `.gitignore` | Arquivo de configuração para ignorar arquivos desnecessários no Git. |
